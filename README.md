# 🛒 ElectroCart

**ElectroCart** is a modern e-commerce web application for purchasing electronic gadgets like phones, laptops, smartwatches, earphones, and more. It includes features like user authentication, shopping cart functionality, and order tracking.

## 🔗 Live Demo

[Click here to view the live project]( http://localhost:8080/)

---

## 🚀 Features

- User signup and login system (email + password)
- Browse and filter products by category
- Add/remove items to/from cart
- Place orders and view order history
- Responsive design for mobile and desktop
- Simple and modern UI

---

## 🧰 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (component library)

---
